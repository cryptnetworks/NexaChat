# Terminology

- **Instance:** one deployed installation.
- **Community:** an independently administered communication space.
- **Category:** an organizational container for spaces.
- **Space:** a text, voice, video, announcement, or forum-like destination.
- **Member:** a person participating in a community.
- **Role:** a named collection of permissions.
- **Direct conversation:** private communication between members.
- **Invite:** a controlled way to join a community.
- **Administration console:** instance or community configuration and moderation interface.

Use these terms in product text, APIs, schemas, documentation, and tests.
